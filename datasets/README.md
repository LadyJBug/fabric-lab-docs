# Northstar Sample Datasets

This folder contains sample datasets used for Microsoft Fabric, Power BI, and data engineering learning scenarios.

## Dataset Files

| File | Description |
|--------|-------------|
| Northstar_Merchants.csv | Merchant reference data containing merchant attributes and details. |
| Northstar_Transactions_part1.csv | Transaction data (Part 1). |
| Northstar_Transactions_part2.csv | Transaction data (Part 2). |
| Northstar_Transactions_part3.csv | Transaction data (Part 3). |

## Scenario

These datasets represent a fictional payment-processing and merchant ecosystem that can be used for:

- Data ingestion into a Fabric Lakehouse
- Data Factory pipelines
- Dataflows Gen2
- Data transformation and cleansing
- SQL analytics
- Power BI reporting and dashboards
- Data science and machine learning exercises

## Recommended Usage

1. Upload all transaction files and merchant data into your Fabric Lakehouse.
2. Combine the transaction files into a single dataset.
3. Create relationships between transaction records and merchant data.
4. Build analytical reports and dashboards to explore:
   - Transaction volume
   - Merchant performance
   - Revenue trends
   - Customer spending patterns
   - Fraud detection scenarios

## File Format

All datasets are provided as CSV files and are intended for educational, demo, and lab purposes.
